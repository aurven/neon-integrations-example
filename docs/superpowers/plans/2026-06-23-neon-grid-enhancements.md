# Neon Grid Enhancements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix the actions dropdown timing bug, add a type column with Lucide icons, and add a date/user info renderer that shows who did what and when — all in the `neon-grid` React widget.

**Architecture:** Three renderer-level changes to `columns.jsx` + minor additions to `NeonGridWidget.jsx` (user cache) and `default.json` (field mapping + new column defs). The `mapNodes` function in `widgets.js` already supports passing full nested objects via `source`; we add `versionInfo` and `lockInfos` to the field list. AG-Grid's dot-notation `field` paths do the rest for nested access.

**Tech Stack:** React 18, AG-Grid Community 31.x, Lucide React (new), Vite 5.x. No new backend changes.

## Global Constraints

- No test runner exists; every verify step uses `npm run build` (syntax/import errors) + a `grep` or manual browser check. Do not add a test runner.
- Inline styles only — no new CSS classes, no new CSS files. Match existing renderer style conventions.
- `lucide-react` must be added to `src/react-widgets/package.json`, not the root `package.json`.
- Build command: `cd src/react-widgets && npm run build`. Output lands in `public/js/react/neon-grid-widget.js`.
- The server uses local HTTPS. All `curl` calls against it need `-k` flag.
- Do NOT modify `print-query-board` files — separate Vite bundle, unrelated.

**Files:**

- Produces: `case 'dateUser':` in `buildColumnDefs` — passes `{ userCache, userField, userAliasField }` via `cellRendererParams`

- Consumes: `row.versionInfo` and `row.lockInfos` — nested objects passed through by new field entries in `default.json`

- [ ] **Step 1: Add getNestedValue helper and DateUserRenderer to columns.jsx**

Add the following after `TypeIconRenderer` (the function added in Task 2) and before `CopyIcon`:

```jsx
function getNestedValue(obj, path) {
  if (!path) return undefined;
  return path.split('.').reduce((o, k) => (o == null ? o : o[k]), obj);
}

function DateUserRenderer({ value, colDef, data }) {
  const userCache = colDef.cellRendererParams?.userCache ?? {};
  const userField = colDef.cellRendererParams?.userField;
  const userAliasField = colDef.cellRendererParams?.userAliasField;

  if (!value) return <span style={{ color: '#9ca3af' }}>—</span>;

  const dateStr = new Date(value).toLocaleString(undefined, {
    day: '2-digit', month: 'short', year: 'numeric',
    hour: '2-digit', minute: '2-digit',
  });

  const userId = getNestedValue(data, userField);
  const userAlias = getNestedValue(data, userAliasField);
  const userName = userId ? (userCache[userId] || userAlias || userId) : (userAlias || null);

  return (
    <div style={{ display: 'flex', flexDirection: 'column', justifyContent: 'center', height: '100%', gap: '1px' }}>
      <span style={{ fontSize: '12px', fontWeight: 600, color: '#3f3c4e', lineHeight: '1.3' }}>{dateStr}</span>
      {userName && (
        <span style={{ fontSize: '11px', color: '#69667f', lineHeight: '1.3' }}>{userName}</span>
      )}
    </div>
  );
}
```
